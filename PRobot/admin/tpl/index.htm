站點數據統計

通過站點統計，您可以整體把握站點的發展狀況。

您還可以查看趨勢統計，把握站點每日變化。

  • 開通空間數: $statistics[spacenum] (趨勢)
  • 全部動態數: $statistics[feednum]
  • 全部日誌數: $statistics[blognum] (趨勢)
  • 全部相冊數: $statistics[albumnum] (趨勢)
  • 全部分享數: $statistics[sharenum] (趨勢)
  • 全部話題數: $statistics[threadnum] (趨勢)
  • 全部評論數: $statistics[commentnum]
  • 開啟應用數: $statistics[myappnum]

程序數據庫/版本

  • 操作系統: $os
  • 數據庫版本: $statistics[mysql]
  • 上傳許可: $fileupload
  • 數據庫尺寸: $dbsize
  • 附件尺寸: $attachsize
  • 當前程序版本: UCenter Home $statistics[version] ( $statistics[release] )
  • UCenter Client 版本: {UC_CLIENT_VERSION} Release {UC_CLIENT_RELEASE}

歡迎光臨管理平台

通過管理平台操作，你可以對發佈的信息進行批量管理。


快捷管理菜單

  • 用戶

    編輯用戶的積分、用戶組、管理空間信息、刪除用戶

  • 事件

    對"我在做什麼"進行批量刪除

  • 日誌

    對日誌進行批量刪除、編輯

  • 相冊

    對發佈的相冊進行批量刪除

  • 圖片

    對發佈的圖片進行批量刪除

  • 評論

    對發佈的評論進行批量刪除

  • 話題

    對群組上面發佈的話題進行批量刪除、精華、置頂

  • 回帖

    對群組上面發佈的回帖進行批量刪除

  • 投票

    對投票進行批量刪除

  • 標籤

    對站點的標籤進行刪除、關閉、合併

  • 群組

    對站點的群組進行刪除、關閉、合併

  • 活動

    對站點的活動進行刪除、審核、推薦

  • 模版樣式

    對站點共享的模版樣式進行批量審核、刪除
