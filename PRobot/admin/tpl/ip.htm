允許訪問的IP列表

只有當用戶處於本列表中的 IP 地址時才可以訪問站點,列表以外的地址訪問將視為 IP 被禁止。
每個 IP 一行，既可輸入完整地址，也可只輸入 IP 開頭。
例如：
"192.168."(不含引號) 可匹配 192.168.0.0～192.168.255.255 範圍內的所有地址，留空為所有 IP 均可訪問。
您當前的IP: $onlineip
